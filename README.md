# playlist-maker-android-RedkinaAlina

Учебный Android-проект Playlist Maker.

## Как запустить

1. Откройте проект в Android Studio.
2. Дождитесь Gradle Sync.
3. Запустите приложение на эмуляторе или устройстве (Run `app`).

## Технологии

- Kotlin
- Android SDK
- Jetpack Compose
- Material 3
